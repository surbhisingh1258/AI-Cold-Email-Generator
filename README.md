# 🚀 AI Cold Email Generator

An AI-powered Cold Email Generator built using the **MERN Stack** and **Groq AI API**. This application helps users create professional, personalized, and high-converting cold emails in seconds by simply providing a prompt.

---

## ✨ Features

### 🔐 Authentication & Security
- User Registration and Login
- JWT-based Authentication
- Password Hashing with Bcrypt
- Protected Routes

### 🤖 AI-Powered Email Generation
- Generate professional cold emails instantly
- Custom prompt-based email creation
- Personalized email content
- Human-like and professional tone

### 📂 Email History
- Save generated emails
- Access previously generated content
- User-specific email records

### 🎨 Modern User Interface
- Responsive Design
- Clean and Intuitive Dashboard
- Tailwind CSS Styling
- Mobile-Friendly Layout

---

### Frontend
- React.js

### Backend
- Node.js
- Express.js
- JWT Authentication

- Groq AI API
